Flexbox

Flex-box用于开发基于弹性盒模型的响应式布局、移动设备优先的 WEB 项目
Github

概述

Flex布局，可以简便、完整、响应式地实现各种页面布局

  1. Flex布局将成为未来布局的首选方案

    已经得到了所有浏览器的支持，
    听说bootstrap 4.0要加入Flex弹性布局哟！

  2. Flexbox知识仅仅将css flex复杂的属性写入更加简单、易用更加语义化的类中，一点都不神秘；
  3. Flexbox基于[容器](container)和[栅格]来布局与bootstrap几乎一致；
  4. Flexbox Github地址：http://github.com/Justisit/Flexbox直接下载
  5. 深入了解css弹性布局将对您使用Flexbox更加顺手、灵活
    学习Flexbox可以很轻松的实现很多复杂的布局
  6. Flexbox是在学习的时候自己做得，可能有一些不如意的地方，希望大家给予鼓励和支持

特点

Flexbox可伸缩的灵活的web页面布局，是未来布局的趋势

  1. Flexbox将复杂难记忆的属性写入简洁明了的样式类中，节省开发时间
    替代以前一些复杂而易错hacks方法(如使用float进行类似流式布局)
    单个类快速实现水平&垂直居中
    轻松实现多栏同高。
  2. 列偏移、嵌套列、列排序
    随意调整[栅格]顺序，（bootstrap中的pull与push）比如让main比sidebar提前渲染出来。
  3. 自动计算间距，实现等宽布局、等距布局
  4. 支持自动填充剩余宽度或高度
    支持item按比例分配宽度或高度
    支持除特定item固定宽度剩余空间其余item按比例分配
    页面内容无法占满一屏的高度时底栏(footer)总是出现在页面的底部
  5. 支持item按样式中定义好的优先级在页面中显示
  6. 丰富的对齐方式，让你像使用photoshop对齐一样操作页面元素位置

列排序

响应式

调整你的浏览器大小，或者用其他设备打开页面，都可以测试这些响应式工具类
.flex-xs-*

同一元素在(&rg;768px)超小屏幕上的显示宽度[栅格]

.flex-sm-*

同一元素在(&rg;992px)小屏幕平板上的显示宽度[栅格]

.flex-md-*

同一元素在(&rg;1200px)中等屏幕桌面上的显示宽度[栅格]

.flex-lg-*

同一元素在(≶1200px)大屏幕桌面上的显示宽度[栅格]
